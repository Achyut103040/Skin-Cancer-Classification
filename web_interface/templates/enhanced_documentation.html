System Overview

The Enhanced Skin Cancer Detection System is a two-stage AI pipeline that combines binary classification with cascade-based fine-grained analysis.

üîÑ Workflow

1
Binary Classification: Determine if lesion is malignant (melanoma) or benign
2
Cascade Classification: If benign, identify specific lesion type (NV, BKL, BCC, etc.)
3
Result Generation: Provide final diagnosis with confidence scores and visual explanations

Model Architecture

üîç Binary Classifier

  • Backbone: ResNet50 (pre-trained on ImageNet)
  • Classifier: Multi-layer MLP with batch normalization
  • Input Size: 224√ó224√ó3
  • Output: 2 classes (Benign, Malignant)
# Binary Classifier Architecture
self.backbone = models.resnet50(pretrained=True)
self.classifier = nn.Sequential(
    nn.Dropout(0.5),
    nn.Linear(2048, 1024),
    nn.BatchNorm1d(1024),
    nn.ReLU(inplace=True),
    nn.Dropout(0.5),
    nn.Linear(1024, 512),
    nn.BatchNorm1d(512),
    nn.ReLU(inplace=True),
    nn.Dropout(0.4),
    nn.Linear(512, 256),
    nn.BatchNorm1d(256),
    nn.ReLU(inplace=True),
    nn.Dropout(0.3),
    nn.Linear(256, 2)
)

Training Process

üìä Data Preparation

  • Source: HAM10000 dataset (8,902 benign images)
  • Split: 80% training, 20% testing
  • Balancing: Oversampling for minority classes
  • Augmentation: Rotation, flipping, color jittering

üéØ Training Strategy

Binary Classifier
  • Loss: CrossEntropyLoss
  • Optimizer: Adam (lr=1e-4)
  • Scheduler: OneCycleLR
  • Epochs: 50
  • Batch Size: 32
Cascade Models
  • Loss: CrossEntropyLoss / FocalLoss
  • Optimizer: Adam (lr=1e-4)
  • Scheduler: OneCycleLR / ReduceLROnPlateau
  • Epochs: Variable (15-50)
  • Cascade Logic: Excludes classified samples

Evaluation Metrics

üìä Primary Metrics

  • Accuracy: Correct predictions / Total predictions
  • Precision: True positives / (True positives + False positives)
  • Recall: True positives / (True positives + False negatives)
  • F1-Score: Harmonic mean of precision and recall
  • AUC-ROC: Area under receiver operating characteristic curve
  • Confusion Matrix: Detailed classification breakdown
  • Confidence Scores: Model prediction certainty
  • GradCAM: Visual explanation heatmaps
